Bienvenue sur le Pokedex !

Le Pokedex est une galerie d'images comprenant 151 images de base.

Vous pouvez intéragir avec la galerie grâce à différentes commandes :

- Add - Pour ajouter une image depuis votre PC dans la galerie (format .jpg ou .png uniquement). 
- Rename - Pour renommer une image présente dans la galerie.
OU
Vous pouvez aussi appuyer sur la touche R de votre clavier
- Remove - Pour supprimer une image présente dans la galerie.
OU
Vous pouvez aussi appuyer sur la touche suppr de votre clavier
- Zoom - Pour augmenter la taille d'une image
OU
Vous pouvez aussi le faire en double cliquant sur une image.
- Exit - Pour quitter la galerie d'image et fermer le programme.

Si vous avez besoin d'aide depuis la galerie le bouton "Help" vous donnera l'assistance nécessaire.

Amusez-vous bien sur le Pokedex !
